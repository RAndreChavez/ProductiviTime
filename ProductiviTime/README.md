# ProductiviTime

ProductiviTime is a productivity and time-management web application I created as my CS50 final project. It was my first real personal programming project and was built using Flask, MySQL, and concepts learned throughout CS50 2022.

The app was designed to help users organize their tasks, schedule important events, and track upcoming deadlines through a simple web dashboard.

## About the Project

ProductiviTime combines multiple productivity tools into one web application:

- User registration and login
- Calendar-based event scheduling
- Countdown timers for upcoming events
- Task creation and organization
- Task status updates
- Editing and deleting tasks
- Event deletion
- A simple dashboard for managing productivity

The goal of the project was to create a practical tool that could help users manage their time, stay organized, and keep track of important deadlines or events.

## Main Features

### User Authentication

Users can create an account, log in, and access their personal productivity dashboard.

### Calendar Events

Users can add events to a calendar and view scheduled events by date.

### Event Countdowns

The app displays countdowns for upcoming events, showing how much time is left until each event.

### To-Do List

Users can create tasks, update their status, edit them, and delete them when they are no longer needed.

### Dashboard

The homepage brings together the main productivity tools in one place, giving users quick access to their calendar, countdowns, and task list.
